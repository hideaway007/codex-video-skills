# “假如你成为 X”空白脚本模板

复制本文件后，把 `{{ }}` 中的内容替换成你的选题信息。

## 选题输入卡

### 1. X 是什么

- 名称：{{X}}
- 类型：{{人物 / 职业 / 阵营 / 身份 / 生物 / 组织成员 / 抽象角色 / 其他}}
- 所属世界或时代：{{现实 / 历史 / 虚构世界 / 行业 / 社会环境}}
- 观众已有认知：{{观众大概知道什么，不知道什么}}

### 2. 观众原本以为

- 最常见爽点：{{成为 X 看起来爽在哪里}}
- 最常见误解：{{观众最容易误会什么}}
- 开场吐槽点：{{可以先拆掉的刻板印象}}

### 3. 奇幻经历

- 第一件离谱的事：{{成为 X 后最先发生的奇幻事件}}
- 最有画面感的能力 / 道具 / 身份变化：{{核心爽点}}
- 最反直觉的世界规则：{{这个世界怎么和观众想的不一样}}
- 最想继续看的奇幻副作用：{{爽点升级后带来的新变化}}

### 4. 上升路径

- 起点：{{你从哪里开始}}
- 入门方式：{{你如何获得资格 / 能力 / 编制 / 身份}}
- 第一次小胜：{{第一次让你觉得自己能行的事件}}
- 第一次不对劲：{{第一次意识到这身份有问题的事件}}

### 5. 碾压力量

- 制度：{{制度压力}}
- 环境：{{自然 / 社会 / 技术 / 战争 / 市场环境}}
- 对手：{{敌人 / 竞争者 / 同僚 / 捕食者}}
- 上级或命运：{{老板 / 组织 / 神明 / 时代 / 客户 / 历史趋势}}
- 绕不开的规则：{{无论怎么选都会改变你的奇幻规则}}

### 6. 结尾命运

- 最终状态：{{成功 / 惨胜 / 死亡 / 被吞噬 / 背锅 / 变成怪物 / 一切只是开始}}
- 回扣开头的反讽点：{{开头爽点最后如何变味}}
- 结尾句：{{这就是 X 的一生 / 一天 / 职业生涯}}

## 正文大纲

### 1. 刻板印象开场

{{先写观众以为 X 很爽的地方，再马上用吐槽拆掉。不要百科化。}}

可用句式：

> 很多人以为成为 {{X}} 约等于 {{表层爽点}}，但说实话，如果你真成了 {{X}}，第一件事大概率不是 {{普通幻想画面}}，而是 {{更离谱的奇幻遭遇}}。

### 2. 本期投放

{{一句话把观众推进第二人称处境。}}

可用句式：

> 本期就让我们把你扔进 {{世界 / 时代 / 系统}}，看看如果你成为 {{X}}，到底会拥有怎样 {{爽点}}、{{荒诞点}}、{{惨点}}，以及很难说算不算奖励的一生。

### 3. 出生 / 入职 / 觉醒

{{从最低起点写起：你醒来在哪里，眼前是什么，马上遇到什么小麻烦。}}

重点：

- 不要先讲大背景。
- 用身体感受、环境压迫、周围人的态度开场。
- 让观众先进入“我在这里”的感觉。

### 4. 第一层错觉

{{你第一次觉得自己拿到好剧本。}}

要写：

- 你得到什么能力 / 资格 / 机会。
- 你第一次赢在哪里。
- 你为什么误以为自己能控制命运。
- 这个爽点会触发什么新能力、新规则或奇幻副作用。

### 5. 规则揭露

{{旁白拆穿这个身份真正的运转方式。}}

要写：

- 这个系统真正看重什么。
- 普通人在里面为什么会被消耗。
- X 的能力为什么既是优势也是枷锁。
- 观众原本误解的地方在哪里。

### 6. 第一次小胜

{{给一个明确爽点，但不要让它太干净。}}

小胜类型：

- 完成任务。
- 打败对手。
- 获得认可。
- 升级身份。
- 活过第一次危机。

小胜后的奇幻变化：

- 更危险的任务。
- 更高阶的能力。
- 更离谱的副作用。
- 更强的阵营或神明注意。
- 更难回头的身份变化。

### 7. 系统加压

{{连续写多个“你以为 A，结果 B”的反转。}}

每 800-1200 字至少一次：

- 你以为这是奖励，结果是债务。
- 你以为这是升职，结果是背锅。
- 你以为这是选择，结果每条路都要死人。
- 你以为你适应了，结果只是变得不正常。

### 8. 大事件卷入

{{把你推入主冲突，让前面规则集中收账。}}

大事件可以是：

- 战争 / 灾难 / 政变 / 清洗 / 暴雷 / 项目失败 / 市场崩盘 / 生态坍塌 / 神明降临 / 客户改需求。

要写出：

- 你为什么躲不开。
- 你之前的小胜为什么反而把你推到中心。
- 这个身份最荒诞的机制如何在此刻爆发。

### 9. 人格变形

{{你为了活下去开始变得不像自己。}}

重点：

- 你理解了以前讨厌的人。
- 你开始使用系统语言。
- 你开始牺牲别人。
- 你开始给自己的变化找理由。
- 你变强了，但也变得更像这个系统需要的东西。

### 10. 反讽收束

{{回扣开头，把最初爽点变成账单。}}

可用句式：

> 所以如果有人问你，成为 {{X}} 爽不爽。爽，当然爽。只是这份爽，从来不是坐在屏幕前想象的那种爽。它更像是一张写着 {{荣耀 / 自由 / 力量 / 成功}} 的账单，翻到背面，全是你已经付出去却再也拿不回来的东西。这就是 {{X}} 的一生。

## 成稿自检

- [ ] 第一段有没有先抓住观众对 X 的刻板印象？
- [ ] 前 500 字内有没有把观众推进“你”的处境？
- [ ] 有没有至少一次明确小胜？
- [ ] 每 800-1200 字有没有认知反转？
- [ ] 知识点有没有变成遭遇，而不是百科介绍？
- [ ] 中段有没有系统压力，而不只是个人倒霉？
- [ ] 后段有没有人格变形？
- [ ] 结尾有没有回扣开头？
- [ ] 最后一句是否能成立为“这就是 X 的一生 / 一天 / 职业生涯”？
