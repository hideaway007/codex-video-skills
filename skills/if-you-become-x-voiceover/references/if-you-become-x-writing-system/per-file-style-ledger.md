# 36 篇逐篇结构账本

本账本不保留大段原文，只记录每篇的结构功能、叙事发动机和可迁移写法。

## 01. 圣血天使

- 发动机：荣耀耗材型 + 悲剧闭环。
- 开场：先塑造高贵、优雅、完美的幻想，再落到受试者的残酷试炼。
- 中段：训练、基因筛选、黑怒、荣耀和慈悲交替出现，爽点始终带诅咒。
- 结尾：回到最初仰望夜空的少年感，把荣耀变成永恒战争与自责。
- 可迁移点：适合写“越高贵越背负诅咒”的身份。

## 02. T850 穿越战锤

- 发动机：理性撞邪型。
- 开场：跨 IP 脑洞 + 免责声明，强调这是脑抽实验。
- 中段：机械逻辑一开始有效，随后被帝国非理性宗教、审判庭和超人战力反复击穿。
- 结尾：从猎手变成实验材料，最后被更高等级人类战士压制。
- 可迁移点：适合写现代技术、AI、理性专家进入荒诞系统。

## 03. 土氏工程师

- 发动机：理性撞邪型 + 工程事故型。
- 开场：正常文明看疯癫帝国，先建立优越感。
- 中段：工程研究逐步深入，越研究越发现“低劣但能运转”的系统不可理解。
- 结尾：实验失控，后来者得到警告。
- 可迁移点：适合写“外行看起来落后，实际背后有不可复制风险”的题材。

## 04. 莫迪安铁卫政委

- 发动机：刻板印象翻案型。
- 开场：先拿“玩具兵”外号和刻板印象开刀。
- 中段：政委幻想英雄路，逐渐理解纪律和固执在极端条件下的价值。
- 结尾：用战场胜利证明被嘲笑的特质正是生存关键。
- 可迁移点：适合写“被笑话的组织特质在极限环境下变成优势”。

## 05. 普通方舟灵族

- 发动机：文明优雅反噬型。
- 开场：精灵、高贵、旅人幻想。
- 中段：道途、感情控制、战争面具和种族伤痛不断加压。
- 结尾：面对屠杀与罪责，优雅文明被迫承认银河规则。
- 可迁移点：适合写“高文明也必须戴上残酷面具”的身份。

## 06. 欧格林人

- 发动机：低智快乐求生型。
- 开场：用力量、忠诚、憨厚建立喜剧。
- 中段：角色理解能力有限，世界荒诞通过误解呈现。
- 结尾：把暴力误解成保护，把危险误解成光明未来。
- 可迁移点：适合写“主角不理解系统，但系统仍利用他”的喜剧题材。

## 07. 千子巫师

- 发动机：知识诅咒型。
- 开场：强大灵能和悲剧战帮并置。
- 中段：知识、灵能、诅咒、背叛不断交织。
- 结尾：智慧没有带来自由，反而加深宿命。
- 可迁移点：适合写“越懂越痛苦”的知识型身份。

## 08. 赎罪远征星际战士

- 发动机：冤罪远征型。
- 开场：指出被迫赎罪者往往不是罪人。
- 中段：资源枯竭、无休战争、内部秩序和复仇意志持续压缩。
- 结尾：完成不可能远征，并把污蔑者送入审判式结局。
- 可迁移点：适合写“被冤枉的人比审判者更能证明制度荒诞”。

## 09. 黑暗灵族

- 发动机：怪物绝望型。
- 开场：用放纵、折磨和灵魂恐惧搭建身份。
- 中段：享乐不是自由，而是延寿和逃避终极吞噬的手段。
- 结尾：极端恶人也在死亡面前崩溃求饶。
- 可迁移点：适合写“越残忍越怕死”的反派身份。

## 10. 海军领主上将

- 发动机：权力囚笼型。
- 开场：巨舰、大炮、海军荣耀。
- 中段：从前线指挥变成行政、政治、盖章和被下属保护。
- 结尾：英雄想上前线，但组织把他当不可损失资产。
- 可迁移点：适合写“升到高位反而失去最初热爱”的职业。

## 11. 导航者

- 发动机：血统诅咒型。
- 开场：高贵、必要、危险的职业。
- 中段：童年特权、社交隔绝、直视亚空间的恐怖并行。
- 结尾：责任压过欲望，个体成为帝国航行工具。
- 可迁移点：适合写“特权不是自由，而是被需要的诅咒”。

## 12. 色孽守密者

- 发动机：怪物自洽型。
- 开场：以美、欲望、艺术和禁忌建立危险吸引力。
- 中段：怪物把腐化理解为引导，把毁灭理解为礼物。
- 结尾：外部调查视角揭示污染不可逆。
- 可迁移点：适合写“反派视角自认仁慈，外部视角恐怖”的题材。

## 13. 纳垢恶魔

- 发动机：怪物慈悲错位型。
- 开场：恶魔作为人类大敌，但从恶魔自身看并非如此。
- 中段：疾病、腐烂、拥抱、安慰被写成一种扭曲关怀。
- 结尾：对花园与父亲的忠诚覆盖了凡人痛苦。
- 可迁移点：适合写“恐怖行为在其价值观中是爱”的身份。

## 14. 兽人猎手政委

- 发动机：被环境同化型。
- 开场：人类逐渐绿皮化的荒诞问题。
- 中段：政委想维持军纪，却逐渐接受当地有效但离谱的规则。
- 结尾：与绿皮式逻辑达成合作，反而形成战场效率。
- 可迁移点：适合写“对抗某种东西久了，自己也开始像它”。

## 15. 雷霆战士

- 发动机：弃子悲歌型。
- 开场：旧时代英雄被历史抛弃。
- 中段：从底层暴力生活到统一战争，再到被替代和背叛。
- 结尾：旧战士再次为了最初信念战斗。
- 可迁移点：适合写“过时英雄被新时代抛弃但仍保留原始忠诚”。

## 16. 行星防卫军政委

- 发动机：参差基层治理型。
- 开场：PDF 是帝国稳定基础，但质量极参差。
- 中段：低分政委面对奇葩星球和基础教育缺失，荒诞变成日常工作。
- 结尾：他成为不断救火的基层建设工具。
- 可迁移点：适合写“基层岗位不是宏大战争，而是无限补窟窿”。

## 17. 坦克指挥官

- 发动机：普通人成为专业老兵型。
- 开场：星界军既是炮灰也是钢铁洪流。
- 中段：农业世界出身、家庭、训练、战车配合、战场经验逐渐积累。
- 结尾：老兵以经验承担更危险目标。
- 可迁移点：适合写“技术岗位的荣耀来自长期默契和风险承担”。

## 18. 蛮骑兵

- 发动机：旧式浪漫撞上未来战争型。
- 开场：历史骑兵史诗铺陈。
- 中段：封建骑兵被星际战争轻视，却在特殊条件下打出奇迹。
- 结尾：老骑兵向末日冲锋，浪漫与死亡重合。
- 可迁移点：适合写“过时技术在极端场景中重新发光”。

## 19. UP 主

- 发动机：现实行业祛魅型。
- 开场：粉丝节点引出“想成为 UP 主”的常见问题。
- 中段：把成功拆成内容、时间、收入、同质化、抗压和运气。
- 结尾：广告收束，形成现实口播的商业插入。
- 可迁移点：适合写“外人看见收益，内部是长期劳动和不确定反馈”。

## 20. 恶魔世界居民

- 发动机：地狱日常化型。
- 开场：帝国很烂，但混沌更无底线。
- 中段：恶魔世界把痛苦、美、欲望和献祭日常化。
- 结尾：外部档案确认整颗世界无药可救。
- 可迁移点：适合写“环境本身把人类底线改写”的社会身份。

## 21. 克里格死亡兵团

- 发动机：原罪赎罪型。
- 开场：人生而有罪，以生命赎罪。
- 中段：训练、战壕、死亡、纪律和少量人性痕迹并存。
- 结尾：死亡成为救赎，冷酷系统里出现一点温柔幻听。
- 可迁移点：适合写“个体价值被压缩成牺牲，但仍留一点人性”。

## 22. 机械教护教军

- 发动机：信仰资产化型。
- 开场：赞美万机之神与机械宗教魔怔感。
- 中段：出生即被扫描、编号、训练，信仰与工具属性合一。
- 结尾：意识被占据，个体变成更高意志的载体。
- 可迁移点：适合写“使命感如何把人变成系统部件”。

## 23. 不可接触者贱民

- 发动机：弱者救赎型。
- 开场：牛马世界里连牛马都不如。
- 中段：极低起点、孤独、被厌恶，能力在危机中显现。
- 结尾：第一次被承认为好人，但立刻成为审判庭资产。
- 可迁移点：适合写“救赎是真的，但自由仍然没有到来”。

## 24. 卡利都司刺客

- 发动机：身份伪装吞噬自我型。
- 开场：用“大姐姐”幻想钩住观众，再拆性别和刺客身份。
- 中段：普通职员生活进入贵族政治，伪装与情感纠缠。
- 结尾：任务完成后，人格与身份残留形成荒诞后日谈。
- 可迁移点：适合写“长期扮演让角色和自我边界变模糊”。

## 25. 猛禽战团

- 发动机：实用主义祛魅型。
- 开场：纠正星际战士只会正面史诗冲锋的刻板印象。
- 中段：隐蔽、侦察、战术耐心压过浪漫战斗。
- 结尾：转入战团档案式后日谈，强调长期战术传统。
- 可迁移点：适合写“真正专业的人不追求好看，只追求有效”。

## 26. 寂静修女

- 发动机：孤独责任型。
- 开场：不可接触者作为反魔终极兵器。
- 中段：强大带来隔绝，责任让她远离普通生活。
- 结尾：帝国新局面开启，但她仍回到牺牲位置。
- 可迁移点：适合写“能力越特殊，越难回到正常人关系”。

## 27. 铸造世界居民

- 发动机：工业耗材型。
- 开场：铸造世界是帝国战争心脏，也是生存下限。
- 中段：家庭、流水线、信仰、工长身份逐步把人磨成零件。
- 结尾：人格涣散，最终被从岗位上取下。
- 可迁移点：适合写“稳定工作如何把人慢慢消耗成机器部件”。

## 28. 绿皮技霸

- 发动机：荒诞创业型。
- 开场：绿皮神话和口语化童话。
- 中段：技霸用绿皮逻辑研究、造船、找目标。
- 结尾：目标变成更大规模的“挖”，以行动欲望收束。
- 可迁移点：适合写“低逻辑但高行动力的角色如何推动世界”。

## 29. 帝国海航

- 发动机：浪漫职业祛魅型。
- 开场：飞行幻想、蓝天、空战浪漫。
- 中段：训练、第一次任务、王牌观察、空中战场危险。
- 结尾：飞行仍浪漫，但每次起飞都可能不回来。
- 可迁移点：适合写“职业浪漫与高死亡率并存”。

## 30. 屁精

- 发动机：底层逆袭喜剧型。
- 开场：绿皮快乐不等于屁精快乐。
- 中段：奴隶、炮灰、保姆、医生等低位职责堆叠。
- 结尾：靠技能和机缘获得位置，但仍处在荒诞生态链中。
- 可迁移点：适合写“最底层角色靠不可替代技能爬上去”。

## 31. 法务部法警

- 发动机：推理祛魅恐怖型。
- 开场：帝国法律巨大低效，引出法警如何执法。
- 中段：案件调查看似推理，实则不断滑向异端与恐怖。
- 结尾：认真处理的案件往往不是推理故事，而是恐怖故事。
- 可迁移点：适合写“外人以为专业工作优雅，内部其实全是高危烂案”。

## 32. 空降兵政委

- 发动机：胆怯到合格型。
- 开场：政委与轨道空降结合，先问会不会吓尿。
- 中段：出生、训练、坠落、混乱战场和政委职责叠压。
- 结尾：丢掉名分，完成职责，成为真正合格者。
- 可迁移点：适合写“角色不是不怕，而是怕到最后仍然跳下去”。

## 33. 帝国贵族

- 发动机：权力囚笼型。
- 开场：观众幻想贵族老爷，反问被骂人渣时怎么办。
- 中段：礼仪、学习、家族、联姻、政治把自由磨掉。
- 结尾：拥有体面人生，但本人承认如果有来生不想再当。
- 可迁移点：适合写“体面生活背后的精致压抑”。

## 34. 萨弗拉化学狗

- 发动机：人渣荣誉型。
- 开场：顶级人渣兵团，先把道德底线踩低。
- 中段：抢劫、分赃、战斗、烂人规则形成独特秩序。
- 结尾：烂人也能以自己的方式获得赦免和荣耀。
- 可迁移点：适合写“低道德群体内部也有规则、忠诚和结局重量”。

## 35. 穿越到战锤宇宙

- 发动机：穿越幻想粉碎型。
- 开场：用猫娘、魔法、精灵等幻想诱导观众。
- 中段：普通现代人进入黑暗宇宙后，常识、身份和生存能力全部失效。
- 结尾：反过来劝人认清自己，不要把逃避现实当成穿越爽文。
- 可迁移点：适合写“爽文前提被现实机制逐条拆掉”。

## 36. 审判官

- 发动机：理想主义腐蚀型。
- 开场：审判官魔怔残酷，引出为何会变成这样。
- 中段：审讯员成长、导师、混沌知识、职责和手段逐渐改变人。
- 结尾：没人天生残酷，但重担会把人塑造成曾经厌恶的样子。
- 可迁移点：适合写“掌权者如何在责任与恐惧中变形”。
