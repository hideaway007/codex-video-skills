## 主要扣分点

1. **压力来源偏单一**
   - 中后段多围绕米湿、数不对、账难看反复推进。
   - 还需要误期、闸口排队、上官催限、临时改道、舟卒逃散、盗匪疑云等不同账单。

2. **历史机制还不够硬**
   - 差帖、里正 / 县吏、保人、签押、勘验、损耗、赔偿、误期等机制没有完全压到“你”身上。
   - 发运司、转运系统、仓场验收等只在边界备注或浅层出现。

3. **大事件收账略单线**
   - 浅滩撞船和仓场挑米成立，但复杂度还不够。
   - 需要加入一次“制度事件”，例如闸口拥堵、临时催纲、文书迟限或上级改道。

4. **人格变形略快**
   - 目前从抗拒到改账、塞钱、混米的变化成立，但更像总结。
   - 需要写成一步步被迫学会：先抗拒，后补记，再主动教新人。

5. **长稿扩展风险**
   - 如果直接拉长，容易重复“损耗 / 背锅 / 账本”同一套压力。
   - 扩写应集中在第 6-10 拍，不要平均加料。
