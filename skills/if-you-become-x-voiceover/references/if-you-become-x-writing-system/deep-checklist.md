# 深度版质检清单

用于检查一篇稿子是否符合 36 篇精读后蒸馏出的 12 拍结构。`checklist.md` 是快速版，本文件是正式长稿版。

## 一、12 拍结构

- [ ] 1. 是否先抓住观众对 X 的幻想。
- [ ] 2. 是否把幻想升级成更值得看的奇幻经历，而不是只泼现实冷水。
- [ ] 3. 是否有明确投放句，把观众推入“你”的身份。
- [ ] 4. 是否从低处出生、底层入局或荒诞落点开始，而不是直接站到顶端。
- [ ] 5. 是否用吃住、身体、工作、等级、费用、歧视等日常细节压入世界观。
- [ ] 6. 是否有第一次能力红利或身份爽点。
- [ ] 7. 这个爽点是否触发新能力、新副作用、阵营任务或命运反噬。
- [ ] 8. 是否写出世界规则荒诞但有效的一面。
- [ ] 9. 是否有大事件把前面的规则集中引爆。
- [ ] 10. 是否出现身份变形或欲望变形。
- [ ] 11. 是否有晚年、档案、后日谈或命运延长。
- [ ] 12. 是否回扣开头幻想完成反讽闭环。

## 二、五种发动机

至少命中一种，最好主次明确。

- [ ] 爽点升级型：越赢越离谱，能力、权限和危险一起升级。
- [ ] 奇幻冒险型：地图、阵营、道具、怪物生态不断展开。
- [ ] 权力迷宫型：贵族、神明、王权、组织和仪式互相牵制。
- [ ] 怪物自洽型：恐怖行为在自身价值观里被解释为爱、秩序、艺术或救赎。
- [ ] 命运反噬型：愿望实现了，但实现方式把你改造成另一种东西。

## 三、语言密度

- [ ] 前 500 字进入“你”的处境。
- [ ] 每 150-250 字有一次“你”的动作、判断或承受结果。
- [ ] 设定说明没有连续 300 字脱离“你”。
- [ ] 每 800-1200 字至少有一次“爽点升级 / 奇幻副作用 / 规则反转”。
- [ ] 每 2000 字左右有一次阶段变化。
- [ ] 结尾前 1000 字进入命运收束。

## 四、口播感

- [ ] 语言有长句推进、插话、转折和现代比喻。
- [ ] 旁白负责拆幻想、补设定和加速，而不是只负责吐槽。
- [ ] 没有故意制造错别字。
- [ ] 没有复制具体创作者的自称、观众称呼、固定口癖或原句。
- [ ] 粗糙但读得懂。

## 五、硬性通过标准

一篇正式稿至少要满足：

- [ ] 12 拍中至少 10 拍清晰存在。
- [ ] 至少命中一种叙事发动机。
- [ ] 至少 5 次“爽点升级 / 奇幻副作用 / 规则反转”。
- [ ] 有一次明确小胜。
- [ ] 有一次世界规则放大。
- [ ] 有身份变形。
- [ ] 有命运后日谈或档案式余波。
- [ ] 结尾回扣开头幻想。
