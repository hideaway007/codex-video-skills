# 反贴脸检查表

本检查表用于防止稿子变成具体创作者的可识别复刻。目标是“同赛道高相似原创口播系统”：结构和读感相似，作者声音必须原创。

## 一、硬性禁止

出现以下任一项，必须返修：

- [ ] 使用具体创作者的自称。
- [ ] 使用具体创作者对观众的固定称呼。
- [ ] 使用具体创作者高辨识私人梗。
- [ ] 复用参考稿原句或近似原句。
- [ ] 使用会让观众误以为是本人写作或本人授权的表达。
- [ ] 把“模仿某人”写进 prompt 目标。
- [ ] 明示或暗示“让读者听不出不是他”。

## 二、贴脸风险评分

每项 0-3 分：

- `0`：没有风险。
- `1`：轻微相似，属于同赛道正常重合。
- `2`：明显接近，需要返修。
- `3`：高度贴脸，必须重写。

### 1. 人称与身份

- [ ] 是否有类似具体创作者的自称。
- [ ] 是否有类似具体创作者的观众称呼。
- [ ] 是否使用了私人经历口吻。

小计：`__/9`

### 2. 口癖与句尾

- [ ] 是否频繁使用某个高辨识口头禅。
- [ ] 是否有固定句尾复用。
- [ ] 是否在转场处形成明显本人腔。

小计：`__/9`

### 3. 梗与段子

- [ ] 是否复用参考稿中的具体笑点。
- [ ] 是否使用特定广告腔或商业插入口吻。
- [ ] 是否使用与选题无关但高度识别的私人梗。

小计：`__/9`

### 4. 句法指纹

- [ ] 是否连续多段都像同一具体创作者在说话。
- [ ] 是否不是“长口播类型相似”，而是“本人腔调换题材”。
- [ ] 是否读完后第一反应是某个具体 UP，而不是这一类内容。

小计：`__/9`

## 三、风险等级

- `0-6`：低风险，可以继续。
- `7-14`：中风险，需要局部替换。
- `15-24`：高风险，需要整段返修。
- `25+`：不可用，重写旁白声音。

## 四、降风险策略

### 1. 替换称呼

把高辨识称呼改成中性称呼：

- `观众`
- `屏幕前的人`
- `很多人`
- `大多数人`
- `想成为 X 的人`

### 2. 替换口癖

不要保留原口癖，改用功能句：

- 拆幻想：`这个想法只看见了奖品，没看见账单。`
- 补设定：`这里的规则不是为了让你好过，而是为了让系统继续运转。`
- 加速推进：`时间就这么过去，带着一堆你没法拒绝的新任务。`

### 3. 替换私人梗

把私人梗改成通用现实比喻：

- 职场烂活。
- 账单。
- 排班表。
- 背锅位。
- 宣传材料。
- 档案记录。
- 绩效和考核。

### 4. 保留结构不保留声音

可以保留：

- 第二人称。
- 爽点转账单。
- 系统荒诞。
- 大事件收账。
- 人格变形。
- 反讽闭环。

必须替换：

- 具体口头禅。
- 具体人设。
- 特定称呼。
- 原句。
- 私人广告腔。

## 五、与其他检查表的关系

使用顺序：

1. `deep-checklist.md`：检查 12 拍结构是否完整。
2. `90-percent-calibration.md`：检查机器指标和人工相似度。
3. `anti-clone-checklist.md`：检查是否过度贴近具体创作者。

如果前两项高分但本表高风险，仍然不能交付。必须降低贴脸风险后再评分。
