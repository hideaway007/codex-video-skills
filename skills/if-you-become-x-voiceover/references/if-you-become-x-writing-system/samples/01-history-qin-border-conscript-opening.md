# 样稿 01：如果你成为秦末戍边征发者

## 选题卡

- 类型：历史人物 / 普通人位置
- 主发动机：弱者荒诞求生型
- 观众幻想：乱世出英雄，胆子够大就能翻身。
- 真实账单：对大多数人来说，乱世不是舞台，是天气、徭役、军法、粮食和户籍一起开的催命单。
- 低处落点：被征发去戍边的普通黔首。

## 1000 字开头样稿

很多人提起秦末，脑子里第一反应大概是乱世、英雄、揭竿而起、王侯将相宁有种乎。听起来很爽，尤其是坐在屏幕前的时候，仿佛你只要胆子够大，胳膊一抬，身边人一喊，时代就会自动给你让出一条通往史书的路。但问题是，乱世对后来写书的人来说叫风云，对真的被扔进去的人来说，通常先叫误期、饥饿、脚底磨烂、以及不知道明天还能不能活着走到下一个亭。

所以本期我们不先让你当皇帝，也不先让你当名将，更不先给你配一群愿意为你赴汤蹈火的兄弟。那太客气了，客气得像一张还没写金额的欠条。我们先把你扔到秦末的路上，成为一个被征发去戍边的普通人。你没有地图，没有兵法，没有龙傲天系统，甚至连自己到底能不能按时赶到都不确定。你有的东西很简单，一身被汗和灰糊住的衣服，一双越来越不像脚的脚，一点不够吃的干粮，还有一条写在律令里的命。

你原本不是想造反的人。说实话，正常人也不会早上起来第一件事就想着我要改变历史。你只是家里被点到了名，户籍在那儿，官吏在那儿，竹简在那儿，谁去、什么时候去、误了怎么办，全都有人替你想好了，唯独没人问你想不想去。你当然也可以说不想，但在这个地方，说不想通常不是一种选择，而是一种让别人更快把你按进规矩里的声音。

出发那天，你以为最难的是离家。母亲给你塞了一点吃的，父亲没有说太多话，邻里有人安慰，说到了边地熬一熬，说不定还能回来。你点头，心里多少还有一点年轻人的侥幸。毕竟故事里的人总要从离家开始，离家之后遇到贵人，遇到机会，遇到一场足以改变命运的大雨。很遗憾，现实里的大雨通常不负责给你机会，它只负责把路冲烂，把粮泡坏，把队伍拖慢，然后让押送你们的人脸色越来越难看。

一开始你还会计算日子。今天走了多少里，明天还能不能补回来，前面那个亭长说的话到底靠不靠谱。后来你就不太算了，因为脚疼会把人的脑子磨得很窄。你开始只关心下一口水在哪里，晚上能不能找个不漏风的地方蜷一下，旁边那个一直咳嗽的人会不会死在路上，死了以后他的东西会不会被收走，还是能不能分你一点。宏大的历史在这个阶段并没有给你任何镜头，时代也没有在你耳边低语。时代只是在催你赶路。

而真正让你开始意识到不对劲的，是你发现所有人都在怕同一件事：不是山路，不是盗匪，不是饿死，而是误期。误期这两个字在你们队伍里像一块压在舌头底下的石头，没人愿意多说，但每个人都知道它的重量。因为在这个系统里，路坏不是理由，雨大不是理由，人病了不是理由，腿断了可能也不是理由。系统不关心你怎么解释，它只关心你有没有按时到。

这时候你终于明白，所谓乱世机会还没来，账单先来了。你甚至还没有见到边塞，没有摸到兵器，没有喊出任何一句能被后世记下来的话，你就已经被一套看不见但压得人喘不过气的规矩推到了悬崖边上。你以为自己走在去戍边的路上，事实上你走在一条没有人负责解释、但所有人都必须付账的路上。
