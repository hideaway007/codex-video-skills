# 样稿 03 返修版：如果你成为边境法师团新兵

- 强化“荣耀耗材型”：天赋越明确，调拨优先级越高。

## 返修后 1000 字开头样稿

很多人提起法师，脑子里大概会自动出现长袍、法杖、古书、火球，以及站在队伍后面优雅抬手，敌人就成片倒下的美好画面。听起来确实不错，尤其是和前排那些拿盾挨揍的人相比，法师这个职业怎么看都像是文明人对野蛮战场的一次降维打击。

但问题是，如果你真的成为边境法师团的新兵，你很快就会发现，所谓远程输出的另一个名字，叫昂贵、脆弱、必须保护、但关键时刻必须顶上去的战略耗材。你不是坐在塔楼里研究星象的神秘人物，你更像一件会头疼、会做噩梦、会因为魔力过载把自己炸开的军需品。

本期就把你扔进北境边墙，看看如果你成为一名刚觉醒魔力的法师新兵，你这一生到底是神秘优雅，还是被军需官、教官、贵族军令和敌方法术一起写进消耗表里的倒霉人生。

你第一次知道自己有魔力，是因为你把家里的水缸冻裂了。很遗憾，这个场景没有吟游诗人唱得那么浪漫。没有白胡子导师从天而降，没有温柔的学院邀请函，也没有什么“孩子，你是被命运选中的人”。有的只是你父亲看着一地碎陶片沉默了很久，你母亲把你拉到身后，而第二天镇上的书记官就带着两名士兵敲开了你家的门。

他们说得很客气。王国感谢你家对边境防务的贡献，你的天赋将得到妥善培养，你会获得比普通农家孩子更体面的未来。听起来很像好事。至少在你十六岁的脑子里，这确实像好事。你甚至想象过自己穿上深蓝色法袍，站在城墙上，伸手召来雷光，下面的人抬头看你，眼里全是敬畏。

然后你被带到边境法师团的训练营，第一件事不是发法杖，而是称重、抽血、登记、测骨龄、测魔力潮汐周期，再把你的名字写进一册厚得像墓碑的名簿里。教官告诉你，从今天开始，你不再只是你自己。你是王国登记在册的初级施法资源，编号、口粮、药剂配给、精神稳定评估和战时调拨优先级都会跟着你走。

你没太听懂，但你大概意识到，原来被命运选中这句话翻译成军方语言，就是你终于有了库存编号。

训练的第一周，你甚至没有学会一个像样的法术。你学的是怎么吃药，怎么睡觉，怎么在头疼时报告，怎么判断自己是不是出现了魔力过载前兆，怎么在听见不存在的钟声时立刻举手。你原本以为法师训练应该是读书、冥想、念咒，结果更多时候像是在学习如何不把自己炸死。教官说这叫基础安全。旁边一个老兵说得更直白：一个新法师最大的价值不是杀敌，而是不要在自己人中间爆开。

你第一次觉得自己真的有用，是在一个月后的靶场。那天风很大，靶子挂在两百步外，普通弩手的准头被吹得一塌糊涂。轮到你时，你按照教官教的方式抬手，念出短咒，掌心里那点冷意顺着骨头爬上指尖，最后化成一枚细小的冰锥飞出去，准确打穿了靶心。周围安静了一瞬，然后有人吹了声口哨。你很难不高兴。毕竟这可能是你离开家之后，第一次觉得自己不是被带走，而是真的被需要。

这当然是好消息。坏消息是，军队最喜欢的就是被需要的人。因为一旦你证明自己能打中靶心，下一步就不是让你回家，而是让你学会在雨里、火里、惨叫里、队友倒在旁边的时候继续打中靶心。你以为自己展示的是天赋，事实上你展示的是可用性。

更坏的消息很快就来了。那天下午，王都来的贵族监察官要看新兵演示，理由是前线近期需要“鼓舞士气的法术展示”。说白了，就是有人想在安全距离内看看王国的钱有没有花出响。教官脸色很难看，因为他知道战场上最危险的从来不只是敌人，还有那些不需要承担后果、但喜欢临时下命令的人。

就在你准备第二次抬手时，北边的烽火塔忽然亮起黑焰。老兵们的表情瞬间变了。你还不知道黑焰代表什么，只看见训练场上所有教官都开始朝武器架跑。直到军号响起来，你才意识到，所谓演示取消了，真正的考试来了。

而边境从来不缺需要被使用的人。
