# 样稿 01 返修版：如果你成为秦末戍边征发者

## 返修目标

- 补上第一次小胜：让“你”不是单纯挨打，而是短暂证明自己有用。
- 把小胜立刻转成账单：有用不等于自由，而是更容易被系统继续使用。
- 加强“系统荒诞”：雨、路、疾病都不算理由，只有误期算结果。

## 返修后 1000 字开头样稿

很多人提起秦末，第一反应大概是乱世、英雄、揭竿而起、王侯将相宁有种乎。听起来确实很爽，尤其是坐在屏幕前的时候，仿佛你只要胆子够大，胳膊一抬，身边人一喊，时代就会自动给你让出一条通往史书的路。

但问题是，乱世对后来写书的人来说叫风云，对真的被扔进去的人来说，通常先叫误期、饥饿、脚底磨烂，以及你明明什么都没做错，却已经快被一套规矩判成有罪。

所以本期我们不先让你当皇帝，也不先让你当名将，更不先给你安排一群愿意为你赴汤蹈火的兄弟。那太客气了。我们先把你扔到秦末的路上，让你成为一个被征发去戍边的普通黔首。你没有兵法，没有地图，没有天命提醒，更没有什么“开局觉醒帝王气”。你有的东西很简单：一身被汗和灰糊住的衣服，一点越来越少的干粮，一双快不像脚的脚，还有一条写在律令里的命。

你原本不是想造反的人。说实话，正常人不会早上起来第一件事就想着我要改变历史。你只是家里被点到了名，户籍在那儿，官吏在那儿，竹简在那儿，谁去、什么时候去、误了怎么办，全都有人替你想好了，唯独没人问你想不想去。你当然也可以说不想，但在这个地方，说不想通常不是一种选择，而是一种让别人更快把你按进规矩里的声音。

出发那天，你以为最难的是离家。母亲给你塞了一点吃的，父亲没有说太多话，邻里有人安慰，说到了边地熬一熬，说不定还能回来。你点头，心里多少还有一点侥幸。毕竟故事里的人总要从离家开始，离家之后遇到贵人，遇到机会，遇到一场足以改变命运的大雨。

很遗憾，现实里的大雨通常不负责给你机会，它只负责把路冲烂，把粮泡坏，把队伍拖慢，然后让押送你们的人脸色越来越难看。

一开始你还会计算日子。今天走了多少里，明天还能不能补回来，前面那个亭长说的话到底靠不靠谱。后来你就不太算了，因为脚疼会把人的脑子磨得很窄。你开始只关心下一口水在哪里，晚上能不能找个不漏风的地方蜷一下，旁边那个一直咳嗽的人会不会死在路上，死了以后他的东西会不会被收走，还是能不能分你一点。

第一次让别人觉得你有用，是因为你记住了前一站驿卒随口说的一句话：前面低洼，雨大时桥会断。队伍里没人愿意多管闲事，押送的人只想按旧路走，因为旧路写在册子上，册子上写了就等于有道理。你硬着头皮提醒了一句。对方先骂你多嘴，后来派人去看，发现桥真的塌了一半。

这算你第一次小胜。队伍少走了一段死路，几个本来快撑不住的人因此多活了一天，押送你们的小吏也第一次正眼看了看你。你心里甚至冒出一点不合时宜的高兴：原来自己不是完全没用，原来脑子在这种地方也能救命。

然而坏消息是，系统最喜欢的就是这种“有点用”的人。从那以后，队伍里一遇到岔路、一遇到天气、一遇到有人病倒，小吏都会下意识看你一眼。你以为小胜会给你一点安全感，事实上它先给你加了一层责任。你说对了，是应该的；你说错了，就是耽误路程。你还没有造反，没有见到边塞，没有喊出任何会被后世记下来的话，就已经学会了一个乱世底层人的第一条规则：有用不是奖励，有用是下一张账单。

而真正让你开始意识到不对劲的，是你发现所有人都在怕同一件事：不是山路，不是盗匪，不是饿死，而是误期。误期这两个字在队伍里像一块压在舌头底下的石头，没人愿意多说，但每个人都知道它的重量。因为在这个系统里，雨大不是理由，桥断不是理由，人病了不是理由，腿断了可能也不是理由。系统不关心你怎么解释，它只关心你有没有按时到。

这时候你终于明白，所谓乱世机会还没来，账单先来了。你以为自己走在去戍边的路上，事实上你走在一条没有人负责解释、但所有人都必须付账的路上。
