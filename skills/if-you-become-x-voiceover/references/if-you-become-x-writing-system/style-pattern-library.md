# 深度句式与段落功能库

本库提供的是功能句式，不是原稿摘抄。使用时替换变量，不要复制具体创作者的人设口癖。

## 一、开场功能句

### 1. 幻想钩子

```text
很多人提起【X】，第一反应大概是【表层爽点】。
```

```text
如果有人告诉你，成为【X】之后可以【幻想 1】、【幻想 2】、【幻想 3】，你大概很难不心动。
```

```text
【X】这东西，听起来就像是【爽文标签】，但问题是，听起来像不代表真进去以后也是。
```

### 2. 纠偏

```text
说实话，这个想法不能说完全错，只能说错得还挺有想象力。
```

```text
很显然，坐在屏幕前想象【X】和真的成为【X】，中间差了大概一整套【制度/环境/命运】。
```

```text
事实上，【X】真正可怕的地方从来不是【表面危险】，而是【真实代价】。
```

### 3. 投放

```text
本期就把你扔进【系统/时代/世界】里，看看如果你成为【X】，到底会得到怎样【爽点】，又要付出怎样【账单】。
```

```text
那么现在，先别急着幻想【表层爽点】，你得先活过成为【X】的第一天。
```

## 二、起点功能句

### 1. 低处出生

```text
你出生了，但很遗憾，这并不是一个值得敲锣打鼓的开始。
```

```text
当你第一次睁开眼的时候，迎接你的不是【幻想画面】，而是【低处环境】。
```

```text
对你来说，人生的第一份资产大概是【小资源】，以及一条暂时还没要你命的命。
```

### 2. 入职 / 入伍 / 入局

```text
你以为自己是被选中了，后来你才明白，被选中很多时候只是系统终于找到了一个可以使用的人。
```

```text
你的名字被写进了名单，这听起来像荣耀，实际上更像账单终于找到了收件人。
```

## 三、小胜转账单

```text
你赢了，这是好消息。坏消息是，从这一刻开始，他们知道你能赢。
```

```text
你第一次觉得自己或许真的适合成为【X】，而系统也第一次确认，你确实值得被继续消耗。
```

```text
奖励当然是有的，只不过奖励的内容通常是下一次更危险的任务。
```

```text
掌声落下以后，真正留下来的不是荣耀，而是更多人看向你的目光。
```

## 四、系统加压

```text
你很快发现，这地方最不缺的就是努力的人，缺的是能被用坏之后还继续运转的人。
```

```text
这个系统并不讨厌你，相反，它相当喜欢你。喜欢到恨不得把每一件烂事都交给你。
```

```text
你以为自己是在往上走，但每往上一层，你离真正的自由就更远一点。
```

```text
事情开始变得荒诞，但更荒诞的是，它确实有效。
```

## 五、怪物自洽

```text
在你看来，这当然不是残忍，这是【X 的价值观】下最正常不过的仁慈。
```

```text
他们的恐惧让你有些困惑，因为你明明是在给予他们【自认为的礼物】。
```

```text
你并不觉得自己堕落了，你只是比他们更诚实地接受了【欲望/秩序/腐烂/暴力】。
```

## 六、理性撞邪

```text
你的判断没有错，至少按照正常世界的逻辑没有错。问题是，这里从来不按正常世界的逻辑结算。
```

```text
你越分析，越觉得这套系统荒唐；但它越荒唐，越能用一种你无法理解的方式继续运转。
```

```text
你第一次意识到，低效不代表无害，愚昧不代表脆弱，离谱也不代表杀不死你。
```

## 七、人格变形

```text
更糟糕的是，你开始理解他们了。
```

```text
你开始用过去自己会厌恶的方式思考问题。
```

```text
你开始给这一切找理由，因为如果它没有理由，那你付出去的东西就全都成了笑话。
```

```text
你确实变强了，只是强到最后，你已经不太像最开始那个会觉得这一切离谱的人。
```

## 八、结尾闭环

```text
所以成为【X】爽不爽？爽，当然爽。只是这份爽背后，写满了你当初没看见的账单。
```

```text
很多年后，你终于站在了当初幻想中的位置，只是你已经不再是当初那个会幻想它的人。
```

```text
档案会记住你的【功绩】，宣传会记住你的【称号】，但你自己记住的，只有【代价】。
```

```text
这就是【X】的一生。
```

```text
这就是成为【X】的代价。
```
