# “假如你成为 X”模板测试矩阵

这份矩阵用于测试模板是否能跨题材稳定工作。每个测试只需要先生成 1000 字开头，再生成 3000 字中段，最后生成结尾，不需要一次写完整篇。

## 测试方法

1. 任选一个测试案例，填入 `blank-template.md`。
2. 用 `prompts/main-generator-prompt.md` 生成开头。
3. 用 `checklist.md` 检查是否前 500 字进入“你”的处境。
4. 继续生成中段，检查是否有系统压力和连续认知反转。
5. 生成结尾，检查是否回扣开头刻板印象。
6. 如果语言太正式，用 `prompts/rough-oralize-prompt.md` 粗糙化。

## 案例 1：历史人物

- X：秦末一个被征发去戍边的普通人。
- 观众原本以为：乱世等于机会，只要胆子大就能翻身。
- 真实代价：你不是站在历史舞台中央的人，你是被天气、徭役、军法和粮食推着走的人。
- 上升路径：从服役、逃亡、入伙，到第一次分到武器。
- 碾压力量：秦法、军粮、地方豪强、起义组织内部秩序、战争消耗。
- 结尾命运：你可能活到改朝换代，但发现新世界也需要有人继续填进旧机器。

通过重点：不要写成秦末百科，要写成一个普通人如何被历史推上赌桌。

## 案例 2：现实职业

- X：刚入行的短视频运营。
- 观众原本以为：懂网感、会蹭热点、爆一条就翻身。
- 真实代价：数据反馈会把你的判断、审美和作息全部重塑。
- 上升路径：实习、剪片、写标题、第一次小爆、被要求复制爆款。
- 碾压力量：平台算法、老板 KPI、客户需求、热点周期、同质化竞争。
- 结尾命运：你终于做出稳定爆款，但开始不知道自己到底喜欢什么。

通过重点：必须写出“爽点是流量，账单也是流量”。

## 案例 3：虚构阵营

- X：奇幻帝国边境法师团的新兵。
- 观众原本以为：法师等于远程输出、神秘、优雅、被人尊敬。
- 真实代价：你本质上是昂贵、脆弱、需要维护的战争耗材。
- 上升路径：觉醒、入团、训练、第一次施法成功、第一次上战场。
- 碾压力量：魔力污染、贵族军令、前线补给、敌方法术、团内派系。
- 结尾命运：你活下来并晋升，但你的学徒开始用当年你害怕的眼神看你。

通过重点：世界观要通过“你正在被部署和消耗”呈现，不要堆设定。

## 案例 4：动物 / 异种身份

- X：城市里一只流浪猫。
- 观众原本以为：自由、可爱、有人投喂、晒太阳。
- 真实代价：自由意味着没有稳定食物、医疗、领地和安全。
- 上升路径：离窝、抢地盘、第一次被投喂、学会避开车和人。
- 碾压力量：天气、疾病、车辆、其他猫、人的善意与恶意、城市更新。
- 结尾命运：你成为这片街区的老猫，但街区拆迁，你熟悉的世界一夜消失。

通过重点：不能只卖萌，必须写出生态位和城市系统如何碾压个体。

## 案例 5：现代社会身份

- X：一线城市里刚买房的普通打工人。
- 观众原本以为：买房等于稳定、安全、人生进入下一阶段。
- 真实代价：稳定本身会变成一种长期支付的判决。
- 上升路径：攒首付、看房、贷款、装修、第一次住进去。
- 碾压力量：房贷、工作稳定性、家庭期待、物业、通勤、市场波动。
- 结尾命运：你终于有了自己的房子，但开始发现自己更像房子的员工。

通过重点：不要写成买房建议，要写成一个人如何被“稳定生活”重塑。

## 通过标准

每个案例都要满足：

- 前 500 字进入“你”的处境。
- 第一段抓住刻板印象并拆掉。
- 至少一次小胜。
- 中段出现系统压力，不只是个人倒霉。
- 至少 5 次认知反转。
- 后段有人格变形。
- 结尾回扣开头。
- 语言保留口播粗糙感，但不故意写错字。
