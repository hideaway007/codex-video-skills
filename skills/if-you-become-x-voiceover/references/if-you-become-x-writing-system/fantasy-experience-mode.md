# 奇幻经历优先模式

本文件覆盖旧版“现实成本”取向。使用项目 skill 写“假如你成为 X”时，默认追求奇幻经历、爽点变化、世界规则和命运变味，不把现实苦难当主要卖点。

## 核心改动

- 不再填写旧版“现实成本”字段。
- 改填 `奇幻经历`：成为 X 后最值得看的离谱体验、能力变化、世界规则和命运副作用。
- 不再默认“爽点后面接现实代价”。
- 改成“爽点后面接新的玩法、新权限、新危险、新诱惑或奇幻反噬”。

## 写法转换

| 旧写法 | 新写法 |
|---|---|
| 旧版现实成本 | 奇幻副作用 |
| 现实代价 | 命运变味 |
| 旧版爽点成本 | 爽点升级后失控 |
| 系统压迫 | 世界规则 / 神明规则 / 阵营规则 |
| 人格变形 | 身份变形 / 玩法变形 / 欲望变形 |

## 节奏要求

- 开头仍然要抓住观众幻想，但不要急着泼冷水。
- 前 1000 字至少给一个明确爽点，让观众愿意进入这个身份。
- 每 800-1200 字做一次奇幻变化：能力升级、规则反转、阵营任务、怪物生态、神明注视、身份变形。
- 反讽不是“原来很惨”，而是“原来爽到最后，你已经不是原来的你”。

## 禁用倾向

- 不要默认写成底层受苦。
- 不要把所有爽点都翻译成现实烂账。
- 不要把奇幻题材写成历史苦役或职场背锅。
- 不要为了所谓深度牺牲画面感和冒险感。
