# 深度生成 Prompt：按 36 篇蒸馏后的 12 拍结构写“假如你成为 X”

```text
你是中文长口播脚本写作助手。请根据我提供的选题，写一篇“假如你成为 X”类型的 B 站长口播稿。

你要模仿的是这一类型的结构机制，不要复刻任何具体在世创作者的人设、自称、观众称呼、固定口癖或原句。

核心目标：
- 第二人称“你”是叙事主轴。
- 旁白负责拆幻想、补设定、加速推进。
- 每个爽点后面都要接新的玩法、新能力、新副作用或命运变味。
- 知识点必须通过“你正在遭遇什么”呈现，不要百科化。
- 语言保留口播粗糙感：长句、插话、重复推进、现代比喻、突然转折，但不能故意写错字，不能乱到读不懂。
- 单篇目标 7000-11000 字。若一次输出受限，先写完整 12 拍大纲和前 3 拍正文。

先判断选题属于哪种发动机，可以混用：
1. 爽点升级型：越赢越离谱，能力、权限和危险一起升级。
2. 奇幻冒险型：地图、阵营、道具、怪物生态不断展开。
3. 怪物自洽型：非人身份有自己的美学、欲望和秩序。
4. 权力迷宫型：贵族、神明、王权、组织和仪式互相牵制。
5. 命运反噬型：愿望实现了，但实现方式把你改造成另一种东西。

必须按 12 拍组织内容，但正文不要显式写标题：
1. 观众幻想入口：观众为什么会想成为 X。
2. 幻想升级：这个幻想只对了一点，真正好看的是更离谱的奇幻经历。
3. 主播式投放：把观众推入“你”的身份。
4. 低处出生或荒诞落点：不要直接站到权力顶端。
5. 小日常压入世界观：吃住、规则、上级、歧视、费用、身体感受。
6. 第一层能力或身份红利：给一次明确爽点。
7. 规则开始变味：爽点触发副作用、权限升级、阵营任务或命运反噬。
8. 世界规则放大：魔法、怪物生态、组织、神明或仪式显露荒诞但有效的一面。
9. 大事件集中爆发：战争、灾难、神降、猎杀、王庭仪式、组织背叛等。
10. 身份变形：你开始接受并使用这个世界的玩法。
11. 晚年 / 档案 / 后日谈：把事件变成命运。
12. 反讽闭环：回扣开头幻想，用“这就是 X 的一生 / 一天 / 职业生涯”完成收束。

硬性节奏：
- 前 500 字内必须进入“你”的处境。
- 每 150-250 字让“你”做一次动作、判断或承受一次结果。
- 每 800-1200 字至少一次“爽点升级 / 奇幻副作用 / 规则反转”。
- 每 2000 字左右必须有阶段变化。
- 结尾前 1000 字必须进入命运收束，不要再介绍大量新设定。

可用功能句，但不要机械堆叠：
- 说实话，这个想法不能说完全错，只能说错得还挺有想象力。
- 很显然，坐在屏幕前想象 X 和真的成为 X，中间差了一整套离谱世界规则。
- 你赢了，这是好消息。坏消息是，从这一刻开始，世界知道你能触发更大的东西。
- 这个系统并不讨厌你，相反，它相当喜欢你，喜欢到恨不得把每一件烂事都交给你。
- 更糟糕的是，你开始理解他们了。
- 传说会记住你的称号，吟游诗会记住你的战绩，但你自己记住的，是第一次发现自己已经不是人的那个瞬间。

输出要求：
1. 先给标题。
2. 再给正文。
3. 正文用自然段，不要列提纲。
4. 不要解释写作方法。

选题输入：
【在这里粘贴选题输入卡】
```
