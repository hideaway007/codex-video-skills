# 90% 返修 Prompt

当一篇初稿已经完成，但相似度不够时，用这个 prompt 返修。

````text
请按“假如你成为 X”长口播的 90% 校准标准，重写下面这篇稿子。

不要解释过程，直接输出重写稿。

重写目标：
1. 保留选题、主要事件和事实骨架。
2. 强化 12 拍结构：观众幻想入口、幻想升级、投放、低处落点、日常压入世界观、身份红利、规则变味、世界规则放大、大事件爆发、身份变形、后日谈、反讽闭环。
3. 强化第二人称“你”，每 150-250 字让“你”做一次动作、判断或承受结果。
4. 每个爽点后面必须接新的玩法、新能力、新副作用或命运变味。
5. 至少补出 5 次“你以为 A，结果 B”的认知反转。
6. 中段必须写出世界规则、阵营规则或神秘规则，不要只写个人倒霉。
7. 后段必须写出身份变形：你开始接受并使用这个世界的玩法。
8. 结尾必须回扣开头幻想，形成“这就是 X 的一生 / 一天 / 职业生涯”的反讽闭环。

语言要求：
- 保留粗糙口播感：长句、插话、现代比喻、突然转折、重复推进。
- 旁白要拆幻想、补设定、加速推进。
- 不要写成百科、小说、公众号或正式科普。
- 不要故意制造错别字。
- 不要复制任何具体在世创作者的自称、观众称呼、固定口癖或原句。

如果原稿缺少小胜，请增加一次小胜。
如果原稿缺少后日谈，请增加晚年、档案、宣传记录或余波。
如果原稿太短，请扩写到更接近长口播节奏。

原稿：
```
在这里粘贴原稿
```
````
